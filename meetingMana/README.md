### 每周菜单页面 
> 环境：webpack2,webpack-dev-server2,babel,sass
> 依赖：vue2   
> 路由：vue-router,异步：vue-resource,图片上传：vue-core-image-upload   
> ui库：weui
#### 命令
> npm run dev 启动开发服务  
> npm run build 合并压缩代码   
#### 分离提示模块
> popnotice模块
> 使用props传递数据，使用$emit自定义事件名称    
#### 搭建可能出现的问题   
> vue-core-image-upload缺少编译依赖，安装babel相关
